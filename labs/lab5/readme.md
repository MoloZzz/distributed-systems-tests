# Locally finite algorithms

Необхідно реалізувати два алгоритми, в обох використовувати розпаралелюваня за допомогою потоків. Реалізації повинні гарантувати відсутність data race.

### Паралельні мінімальні відстані (локально-скінченний Bellman-Ford)

Ідея:

- кожна вершина рахує кандидатні відстані
- оновлення відбувається синхронно (double-buffering)
- бар’єр гарантує узгодженість

---

- немає lock → немає race condition
- double buffer → немає частково оновлених даних
- barrier → синхронні раунди
- після N ітерацій → Bellman-Ford гарантує правильність

### Локально-скінченна мінімізація розфарбування

Ідея:

- кожна вершина оновлюється незалежно
- конфлікти вирішуються через фазу “snapshot + update”
- зупинка: стабілізація

---

- кожна ітерація = повна синхронна фаза
- немає читання “напівоновлених” значень
- мінімізація локальна (тільки сусіди)
- завершення після стабілізації
- немає data race (лише barrier + контрольована зміна)
