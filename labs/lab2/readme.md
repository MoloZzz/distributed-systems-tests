# Пошук MIS у дереві

Реалізувати пошук найбільшої незалежної множини у дереві.
Вхідні дані: дерево (можна згенерувати або зчитувати з файла/захардкодити декілька варіантів дерев для тестування)
Вихідні дані: розмір MIS; індекси верших, що входять до MIS (якщо можливо декілька варівантів MIS, то обрати будь-який).
